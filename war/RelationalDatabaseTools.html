Relational Database Tools

Attribute closure calculator, Candidate key calculator, Minimum (Canonical) cover calculator, Functional dependency calculator and Normal form calculator

Currently supported functionality: (Updated 4/22/2015)
  • Correctly parses user input for relation schema and functional dependencies.
  • Calculates the closure of each element in the power set of given attributes.
  • Calculates minimum-set (candidate) keys and superkeys.
  • Calculates a minimum (canonical) cover of functional dependencies on the given set of functional dependencies.
  • Derives all single-attribute right-hand side functional dependencies.
  • Checks if given relation schema is in First, Second, Third and Boyce-Codd normal forms.
  • If given relation schema is not in 3NF, will decompose input relation in a lossless and functional dependency preserving manner. **Beta feature**
To-do in the future:
  • Allow for BCNF decomposition.
  • Add support for multivalued dependencies.
  • Check if given relation schema is in Fourth Normal form.
